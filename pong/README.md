# Pong

A Pong clone with a basic AI. The player can be controlled with either the keyboard or a controller.

## Demo

The demo shows a game of Pong being played. The player on the left is controlled by the user, while the player on the
right is controlled by the AI. The AI is able to track the ball and move to intercept it. The game ends when the ball
goes off the screen, and in this game the first to 2 points wins.

The video demo is also available [here](./demo.mp4).
